## Quick Start

### Prerequisites
- Python 3.8+
- Node.js 18+
- OpenAI API key or Azure OpenAI credentials

### Setup
```bash
# Clone and install dependencies
git clone <repo-url>
cd GAIK-platform

# Install dependencies
pnpm install

# Start development
pnpm dev
```
